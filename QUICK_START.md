## 🏠 Complete User Journey (5 Minutes)

### As a Tenant

1. **Land** → Landing page with hero search
2. **Search** → Set filters (Mombasa, KSh 5k-10k, 1 Bed)
3. **Browse** → See 8 verified properties with ratings
4. **Detail** → Click any property → See full info, landlord rating, matatu routes
5. **Chat** → Start conversation with landlord
6. **Pay** → View M-PESA payment instructions
7. **AI Help** → Click chatbot for instant assistance

### As a Landlord

1. **Login** → Use `juma@example.com`
2. **Dashboard** → See property stats, views, chats
3. **Add Property** → Click "+ Add Property"
4. **4-Step Wizard**:
   - Step 1: Title, description, price, bedrooms
   - Step 2: Location, photos
   - Step 3: Amenities, security features
   - Step 4: M-PESA till, preview
5. **Submit** → Property sent for verification
6. **Chat** → Respond to tenant inquiries

### As Admin

1. **Login** → Use `admin@housecom.co.ke`
2. **Dashboard** → View platform stats
3. **Pending Tab** → See properties awaiting approval
4. **Approve/Reject** → Moderate listings
5. **Users Tab** → Manage user accounts

## 🐛 Troubleshooting

### Issue: Nothing appears after login
**Fix**: Check browser console, refresh page. Mock data loads from localStorage.

### Issue: Chatbot not responding
**Fix**: Click again - animation might be slow. Check for typos in queries.

### Issue: Images not loading
**Fix**: Using Unsplash CDN - check internet connection.

### Issue: OTP not working
**Fix**: ANY 6 digits work in demo mode (e.g., 123456).
